# Plan 01-01: BMS Kapitel Audit & Qualitaetsverbesserung

## Objective

Alle BMS-Kapitel auf konsistente Struktur, Lesbarkeit, Verstaendlichkeit und Tiefe pruefen und verbessern. Jedes Unterkapitel muss dem Template aus CLAUDE.md folgen.

## must_haves

- Jedes BMS-Unterkapitel hat konsistente Struktur (Einleitung, "In diesem Kapitel lernst du:", Kern, Uebergaenge, Merksaetze, MedAT-Fokus, Zusammenfassung)
- Alle Unterkapitel haben mindestens 800 Woerter
- Merksaetze (> **Merke:**) in jedem Unterkapitel vorhanden
- Stichworte-Array pro Unterkapitel gefuellt
- npm run build fehlerfrei

## Tasks

<task id="1" title="Audit aller BMS-Kapitel auf Qualitaetsmetriken">
Erstelle ein Audit-Skript oder fuehre manuell durch:
- Pro Unterkapitel: Wortanzahl, Merksatz-Count, Stichworte-Count, Struktur-Check (Einleitung/Lernziele/Zusammenfassung vorhanden?)
- Ergebnis: Liste aller Unterkapitel unter 800 Woerter, ohne Merksaetze, ohne Stichworte
- Fokus auf: Chemie (11/19 unter 800w), Physik (13/19 unter), Mathematik (17/25 unter)
- Biologie ist Referenzqualitaet — als Template nutzen
</task>

<task id="2" title="Chemie-Kapitel auf Template-Niveau bringen" depends_on="1">
Fuer jedes Chemie-Unterkapitel unter 800 Woerter oder ohne Merksaetze:
- Inhalt erweitern auf mindestens 800-1200 Woerter
- Struktur anpassen: Einleitung → Lernziele → Kern → Merksaetze → MedAT-Fokus → Zusammenfassung
- Uebergangssaetze zwischen Abschnitten einfuegen
- Merksaetze als `> **Merke:**` Blockquotes
- Stichworte-Array befuellen
- Format: 30% Prosa / 55% Bullets / 15% Tabellen (AMBOSS-Stil)
</task>

<task id="3" title="Physik-Kapitel auf Template-Niveau bringen" depends_on="1">
Gleiche Verbesserungen wie Task 2 fuer alle Physik-Unterkapitel.
- 13/19 Unterkapitel unter 800 Woerter
- 14 ohne Merksaetze
- Formeln korrekt in Markdown
- Einheiten und SI-System konsistent
</task>

<task id="4" title="Mathematik-Kapitel auf Template-Niveau bringen" depends_on="1">
Gleiche Verbesserungen wie Task 2 fuer alle Mathematik-Unterkapitel.
- 17/25 Unterkapitel unter 800 Woerter
- 18 ohne Merksaetze
- Formeln und Rechenbeispiele ergaenzen
- Loesungswege Schritt fuer Schritt
</task>

<task id="5" title="Biologie-Kapitel Feinschliff" depends_on="1">
Biologie ist bereits Referenzqualitaet, aber:
- Pruefen ob alle Stichworte abgedeckt
- Fehlende Uebergangssaetze ergaenzen
- Konsistenz der Merksatz-Formatierung pruefen
</task>

<task id="6" title="Build-Verifikation und Audit" depends_on="2,3,4,5">
- npm run build
- npm run typecheck
- npm run audit-bms (0 Fehler)
- npm run audit-images
- Stichproben-Check: 2 Unterkapitel pro Fach auf Lesbarkeit
</task>
